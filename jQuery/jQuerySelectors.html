Can use a $("cssLocator") to selected elems.

e.g.$("#delicious")

Can also use things to modify the elements in jquery.

such as: $("h1").css("color","yellow"), as opposed to css style of document.querySelector("h1").style.color="orange"

Additionally, an object can be created to hold all necessary styles in the above which can then be used to manipulate any of the css props we want

var styles = { color:"red",background:"pink",border:"2px solid purple"}
$("h1").css(styles)

Multiple elements can be selected via the jQuery in a single line as well.

This will turn all list elements green in a single line without the use of a loop.
$("li").css("color","green")

Multiple elements can also take multiple styles in a single line as well.

$("li").css({fontSize: "20px", border: "3px dashed purple", background: "rgba(20,67,10,0.5)"})
