    .attr()

    $('img').css("width") <-- We just want to read the image width to start

    $('img').css("width","200px") <-- Shrink it to manage

    $('img').attr("src","https://c3.staticflickr.com/3/2418/2243463214_f32ab004af_b.jpg") <-- Murder Death ball five thousand

    $("input").attr("type","color")

    $("input").attr("type","checkbox")

    $("input").attr("type","text") <-- back to text since we 'll use it in the video

    Alternatively to changing all 3 images of the pine martens to murder death balls five thousand, we can just change one of them using these options

    $("img:first") | $("img:first-of-type").attr("src","new LInk of Death")

    Alternatively, you can use the last attribute for the last one --> $("img").last().attr("src","new URL Of death")

    .val()

    This method will work on all elements that contain a value attribute.

    $("input").val()

    $("input").val("Some New Magics Text")

    $("select").val()

    .addClass() | .removeClass() | .toggleClass()

    $("h1").addClass("correct")

    $("h1").removeClass("correct")

    $("li").addClass("wrong")

    $("li").removeClass("wrong")

    $("li").addClass("correct)

    $("li").toggleClass("correct")

    $("li").first().toggleClass("done")
